# 커스텀 밈 만들기

사용자가 원하는 밈을 조사→작성→검증→저장하는 파이프라인을 실행합니다. 커스텀 밈은 플러그인 업데이트와 무관하게 유지됩니다.

## 실행

1. 밈 이름이 없으면 어떤 밈을 만들지 물어봅니다.

2. 밈 이름이 있으면:

### Step 1: 적합성 조사

`$ARGUMENTS`에 대해 웹 검색으로 조사합니다.

**조사할 것:**
- 밈의 유래와 출처
- 대표 표현 10개 이상
- 말투 특성 (톤, 문장 구조, 특수 어미)
- 대중적 인지도

**적합성 판단 기준:**
- 말투 변환이 가능한가? (어휘 변환 사전 15개 이상 만들 수 있는지)
- 개발 상황 + 일상 대화 모두 적용 가능한가?

**부적합 시 즉시 중단:** 단순 유행어, 챌린지/댄스, 이미지/짤 밈은 말투 변환이 안 되므로 사용자에게 이유를 설명하고 대안을 제안합니다.

### Step 2: 밈 파일 작성

기존 밈 파일을 참조해서 동일한 품질로 작성합니다:

```
!`cat "${CLAUDE_PLUGIN_ROOT}/memes/kim-dong-hyun.md"`
```

**필수 구조:**

```markdown
---
name: <kebab-case-id>
year: <유행 연도>
description: <한 줄 설명>
---

# 밈 이름
1~2문장 설명

## 유래
출처와 배경

## 어휘 변환 사전
| 일반 표현 | 밈식 표현 |
15개 이상 항목

## 말투 규칙
6개 내외. 각 규칙에 세부 매핑 포함

## 변환 방식
1. 원래 문장의 맥락(장소, 상황, 대상) 그대로 유지
2. 감정/평가 표현만 밈 어휘로 교체
3. 자연스러운 문장으로 출력 (낱개 어록 나열 금지)
4. 변환된 결과만 출력

## 예시
8개 이상 입력→출력 (개발 + 일상 혼합)
### 입력: "..."
### 출력:
...
```

### Step 3: 품질 검증

작성한 파일을 아래 체크리스트로 검증합니다. 미달 시 직접 수정합니다.

- [ ] frontmatter에 name, year, description 있음
- [ ] 어휘 변환 사전 15개 이상
- [ ] 말투 규칙 6개 내외 + 세부 매핑
- [ ] 변환 방식 4단계 포함
- [ ] 예시 8개 이상 (### 입력: / ### 출력: 형식)
- [ ] 예시가 개발 + 일상 상황을 골고루 포함
- [ ] 어휘 사전 표현이 예시에서 실제 사용됨
- [ ] 변환된 문장이 해당 인물/캐릭터가 할 법한 말인지

### Step 4: 저장

커스텀 밈 디렉토리에 저장합니다:

```
!`mkdir -p "${CLAUDE_PLUGIN_DATA}/custom-memes"`
```

파일 경로: `${CLAUDE_PLUGIN_DATA}/custom-memes/<id>.md`

### Step 5: 안내

- 저장 완료를 알리고
- `/k-meme:on <id>`로 바로 사용 가능하다고 안내
- `/k-meme:list`에서 [커스텀] 태그로 표시된다고 안내
