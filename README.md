# k-meme 🇰🇷

한국 인터넷 밈의 말투로 Claude의 어투를 바꿔주는 [Claude Code](https://claude.ai/code) 플러그인입니다.

김동현의 세컨드 말투, 장원영의 럭키비키, 유병재의 건조체 등 **22가지 한국 밈 화법**을 지원합니다.

## 설치

Claude Code에서 아래 명령어를 순서대로 입력하세요:

```bash
# 1. 마켓플레이스 추가 (처음 한 번)
/plugin marketplace add mylumiere/k-meme

# 2. 플러그인 설치
/plugin install k-meme@k-meme

# 3. 플러그인 활성화
/reload-plugins
```

업데이트:
```bash
/plugin update k-meme@k-meme
/reload-plugins
```

## 사용법

```bash
# 밈 목록 보기 (최신순 정렬)
/k-meme:list

# 밈 활성화 (즉시 어투 적용)
/k-meme:on kim-dong-hyun

# 밈 상세 설명 보기
/k-meme:help lucky-vicky

# 밈 비활성화 (원래 어투로 복귀)
/k-meme:off
```

한 번에 하나의 밈만 활성화됩니다. 새 밈을 켜면 기존 밈은 자동으로 교체됩니다.

## 밈은 어떻게 작동하나요?

각 밈은 Claude의 **대화 어투만** 바꿉니다. 코드 작성, 파일 편집 등 기술적 작업은 정상적으로 수행되며, 설명이나 대화 부분에서만 밈 화법이 적용됩니다.

밈 파일에는 **어휘 변환 사전**(~15개 표현 매핑), **말투 규칙**(톤, 문장 구조), **변환 방식**(맥락 유지 원칙), **입력→출력 예시**(8개+)가 포함되어 있어 Claude가 자연스럽게 말투를 변환할 수 있습니다.
